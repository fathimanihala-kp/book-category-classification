# 📚 Book Category Classification

## 🔹 Project Overview
This project uses machine learning algorithms to classify books into categories such as novels, cooking, and religious texts.

## 🔹 Models Implemented
- Logistic Regression
- MultinomialNB
- DummyClassifier
- KNeighborsClassifier

## 🔹 Results & Insights
- Logistic Regression achieved the highest accuracy
- The Multinomial NB takes less time

## 🔹 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 🔹 How to Run
1. Open the notebook in Google Colab or Jupyter
2. Run all cells step by step
