# 📚 Book Category Classification

## 🔹 Project Overview
This project uses machine learning algorithms to classify books into categories such as novels, cooking, and religious texts.

## 🔹 Models Implemented
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## 🔹 Results & Insights
- Logistic Regression achieved the highest accuracy
- KNN provided the fastest execution time

## 🔹 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

2. Run all cells step by step
